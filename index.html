Containers

A container is a centered div with a max-width of 1140px. This can be modified with the $container-width variable. It is applied using the container class. This entire page is inside a container.

<div class="container">
	<p>Content goes here</p>
</div>

Grids

Grids can be created with the row and column classes. Each column will be the same size so in a row with 2 columns, each column will be 50% wide.

A column
Another column
<div class="row">
	<div class="column">A column</div>
	<div class="column">Another column</div>
</div>

You can additionally add the column-1 through column-8 classes to define the size of a column. By default there are 8 sizes but this can easily be changed with the $columns-max variable.
