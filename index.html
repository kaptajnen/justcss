JustCSS

A small CSS UI framework for fast prototyping without the fuss.

Containers

A container is a centered div with a max-width of 1140px. This can be modified with the $container-width variable. It is applied using the container class. This entire page is inside a container.

<div class="container">
	<p>Content goes here</p>
</div>

Grids

Grids can be created with the row and column classes. Each column will be the same size so in a row with 2 columns, each column will be 50% wide.

A column
Another column
<div class="row">
	<div class="column">A column</div>
	<div class="column">Another column</div>
</div>

You can additionally add the column-1 through column-8 classes to define the size of a column. By default there are 8 sizes but this can easily be changed with the $columns-max variable.

Forms

Forms have some basic styling that do not require any additional classes to be applied.

A button as an achor element 
<form>
	<label for="input">Text field</label>
	<input type="text" id="input" />
	<input type="submit" value="Submit" />
	<button type="button">I am a button</button>
	<a href="#" class="button">A button as an achor element</a>
	<div>
		<label><input type="radio" name="radio" /> Radio 1!</label>
		<label><input type="radio" name="radio" /> Radio 2!</label>
	</div>
	<input type="date" />
	<select>
		<option>Option 1</option>
		<option>Option 2</option>
		<option>Option 3</option>
	</select>
</form>

Messages

A message can be used to show important information after an action is performed. It is applied using the message class.

This is a message

<div class="message"><p>This is a message</p></div>
